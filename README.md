# Hack Your First 10 Machines

You’ve learned the tools in [Start Hacking Today: The Only 3 Tools You Need](https://github.com/randalltr/start-hacking-today).

Now it’s time to actually hack.

This book walks you through your first 10 Hack The Box machines using a simple, repeatable process:

```
    ┌──────────────┐
    │     Nmap     │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │   Exploit    │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │   PrivEsc    │
    └──────┬───────┘
           │
           ▼
    ┌──────────────┐
    │     root     │
    │     id=0     │
    └──────────────┘
```

No fluff. No overwhelm. Just reps.

If you can finish these 10 machines, you are no longer a beginner.
